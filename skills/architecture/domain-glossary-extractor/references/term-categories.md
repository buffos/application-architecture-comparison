# Term Categories

Use these categories when extracting terms.

## Actors

Examples:

- sales clerk
- manager
- warehouse clerk
- administrator

## Business Objects

Examples:

- quote
- order
- shipment
- return request

## Workflows or Actions

Examples:

- submit for approval
- convert to order
- capture payment
- ship order

## Policies or Rules

Examples:

- discount policy
- return policy
- approval policy

## States or Statuses

Examples:

- draft
- approved
- backordered
- shipped

## Metrics or Reports

Examples:

- low stock
- conversion rate
- return rate

## External Systems or Boundary Terms

Examples:

- payment provider
- shipping carrier
- plugin
